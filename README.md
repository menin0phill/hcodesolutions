# H-code Solutions - Portfolio Landing Page

Este é o site oficial de portfólio (landing page) da **H-code Solutions**. O projeto foi desenvolvido com foco em estética premium (tema escuro e detalhes metálicos dourados), performance extrema de carregamento e legibilidade impecável das informações.

## 🚀 Tecnologias Utilizadas

- **HTML5 Semântico**: Estruturação limpa e otimizada para SEO.
- **CSS3 Vanilla**: Estilização moderna com variáveis, layouts em Grid/Flexbox, efeitos de vidro (*glassmorphism*) e sombras brilhantes responsivas.
- **JavaScript (ES6+)**: Lógica de interação da interface (Intersection Observer para scroll reveals, cursor glow reativo e menu mobile).
- **HTML5 Canvas 2D API**: Sistema de partículas e constelação digital interativa no Hero, rodando de forma 100% nativa e livre de travamentos.
- **GSAP (GreenSock)**: Otimização de transições e scroll suave.
- **SVG Vetorial**: Logo responsiva em alta definição com gradiente metálico dourado personalizado.

## 📦 Estrutura de Arquivos

```text
├── index.html          # Estrutura principal da página
├── styles.css          # Estilização visual completa e responsiva
├── app.js              # Lógica de interações gerais e animação do canvas 2D
├── logo.svg            # Logo vetorial oficial em alta definição
└── .gitignore          # Arquivos ignorados pelo Git
```

## 🛠️ Como Executar Localmente

Como o projeto é construído em cima de tecnologias web nativas sem dependências complexas de compilação, você pode executá-lo de duas formas simples:

### Opção 1: Servidor Estático Simples (Recomendado)
Você pode usar qualquer servidor de desenvolvimento para servir os arquivos estáticos:

**Usando Node.js / npx:**
```bash
npx http-server -p 8080
```

**Usando Python:**
```bash
python -m http-server 8080
```
Depois, basta acessar `http://localhost:8080` no navegador.

### Opção 2: Abertura Direta
Você também pode simplesmente dar um duplo clique no arquivo `index.html` para abri-lo diretamente no navegador.
